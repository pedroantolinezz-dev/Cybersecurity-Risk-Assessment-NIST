# Análisis de Riesgos de Ciberseguridad - Lumina Health Solutions

Este proyecto presenta una evaluación completa de riesgos para una empresa del sector HealthTech, utilizando el framework **NIST CSF** y estándares **ISO 27001**.

##  Objetivos
* Identificar activos críticos de información.
* Evaluar amenazas y vulnerabilidades (Riesgo Inherente).
* Proponer controles técnicos y administrativos para la mitigación.

##  Contenido del Repositorio
* **Informe Ejecutivo (PDF):** Documentación detallada del alcance, metodología y conclusiones.
* **Matriz de Riesgos (Excel):** Cálculos de impacto, probabilidad y plan de tratamiento.

## Frameworks Utilizados
* **NIST Cybersecurity Framework (CSF)**
* **ISO/IEC 27001**
